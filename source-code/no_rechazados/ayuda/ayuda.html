Información previa a darse de alta como usuario

No-rechazados es un sitio donde el usuario puede poner en comun proyectos artísticos. Se entiende por proyectos artísticos, aquellas producciones culturales que se encuadran en la definicion de artes mayores, quedando excluidas las artesanias. Esta junto a que las obras deven haverse presentado primeramente a concurso de una convocatoria artística son las únicas condiciones para poder introducir proyectos en el sistema.

No-rechazados no se responsabiliza del uso que los usuarios den de este. Igualmente todas las obras y contenidos introducidos en él están bajo licencia de creative commons.Para ver información sobre la licencia: http://creativecommons.org/licenses/by-nc-sa/2.5/es Esta licencia no protege las obras que se introduzcan de posibles copias o mal uso de estas, pero asegura, aclara, que la obras contenidas tienen un propietario y que no se puede hacer un uso comercial de este sin consentimiento de este.

No-rechazados se compromete a un buen uso de los datos personales de los usuarios inscritos al servicio, por lo que todas las obras quedan bajo el anonimato de un pseudónimo o “nick”. Los visitantes, usuarios no pueden conocer la dirección de correo electrónico de los autores de los proyectos introducidos. Para contactar con estos se hará a través del formulario de contacto con el administrador, después este se pondra en contacto con el autor para que avisarle sobre que hay alguien interesado en contactar con él.

Los datos requeridos para hacer uso del servicio son:

nombre de usuario: este sera el nombre que se usará para nombrar al autor de las obras y para las comunicaciones con el administrador. Es muy importante introducir un nombre definitivo porque es el único dato personal que no se puede cambiar en el futuro. Este campo tiene de taner un mínimo de 6 carácteres.
Pasword: la contraseña para poder entrar en el sistema y  así poder introducir proyectos o concursos. Tiene que tener una tamaño mínimo de 6 carácteres.
Mail: la direccion de correo electronico para una comunicación entre la administracion y el usuario. A este mail se enviarán los datos recordatorios del usuario y la pasword.
Nombre y apellidos: campo fundamental para dar a las obras contenidas un autor. No se usará en la web por motivos de mantener el anonimato de los usuarios.

No-rechazados se reserva el derecho de eliminar los proyectos introducidos si considera que estos no cumplen los requisitos para poder estar dentro de la web (que sea una proyecto artístico y que haya sido presentado primeramente a un concurso), o si las definiciones introducidas sobre el proyecto son incompletas o carecen de sentido.
