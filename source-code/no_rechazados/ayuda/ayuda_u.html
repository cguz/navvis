Información previa a darse de alta como usuario

No-rechazados es un lugar donde el usuario puede introducir y poner en común proyectos artísticos. Se entiende por proyectos artísticos, aquellas producciones culturales que se encuadran dentro de la definición de artes mayores (pintura, escultura, video, fotografia, met-art...), quedando excluídas las artesanias. Esta definición junto a que las obras deven haverse presentado primeramente (y ser rechazadas) a un concurso o convocatoria artística, son las únicas condiciones para poder introducir proyectos en el sistema.

No-rechazados no se responsabiliza del mal uso que los usuarios hagan del servicio.

Todas las obras y contenidos introducidos en él están bajo licencia de creative commons. Esta licencia no protege las obras que se introduzcan de posibles copias, pero aclara al posible visitante, que la obras contenidas tienen un autor/es y que no se puede hacer un uso comercial de estas sin consentimiento del propietario.

No-rechazados se compromete a un "buen uso" de los datos personales de los usuarios inscritos al servicio. Todas las obras quedan bajo el anonimato de un pseudónimo o “nick”. Los visitantes, usuarios no pueden conocer la dirección de correo electrónico de los autores de los proyectos introducidos. Para contactar con estos se hará a través del formulario de contacto con el administrador, después este se pondrá en contacto con el autor para avisarle que hay alguién interesado en contactar con él.

Acontinuación se detalla los datos requeridos para hacer uso del servicio, y para qué son necesarios estos:
nombre de usuario: este será el nombre que se usará para nombrar al autor de las obras y para las comunicaciones con el administrador. Es muy importante introducir un nombre definitivo porque es el único dato personal que no se puede cambiar en el futuro. Este campo tiene de tener un tamaño mínimo de 6 carácteres.
Pasword: la contraseña para poder entrar en el sistema y  así poder introducir proyectos o concursos. Tamaño mínimo de 6 carácteres.
Mail: la dirección de correo electrónico para una comunicación entre la administración y el usuario. A esta dirección se le enviarán los datos recordatorios del usuario y la pasword.
Nombre y apellidos: campo fundamental, para especificar que las obras contenidas tienen un autor. Es recomendable poner el nombre correctamente. No será visible en la web para mantener el anónimato de los usuarios.

Aquellos que deseen que sus datos reales personales aparezcan en la web pueden introducirlos libremente cuando introduzcan un proyecto incluyendolos en la descripción de este.

No-rechazados se reserva el derecho de eliminar los proyectos introducidos si considera que estos no cumplen los requisitos para poder estar dentro de la web (que sea una proyecto artístico y que haya sido rechazado por un concurso), o si las definiciones introducidas sobre el proyecto son incompletas o carecen de sentido. Se asegura que No rechazados no eliminará proyectos usando criterios de calidad artísticas, ya que precisamente es esta la idiosincracia del proyecto: crear un espacio expositivo virtual donde se discuta sobre el criterio de selección.
