Ayuda introducción de convocatoria.


Esta sección tiene como objetivo dar al usuario la posibilidad de poner en común información sobre convocatorias artísticas, que estan aún vigentes y describir la convocatoria a la que se ha presentado un proyecto.
La descripción de este concurso se hace a través de los siguientes campos:
Nombre: aquí se introducirá el nombre de la convocatoria, este nombre es un campo único por lo que sólo se pueden introducir nuevas convocatorias que no coincida el nombre con convocatorias anteriormente introducidas. Esto se ha realizado de esta manera para permitir una busqueda al visitante de los proyetos introducidos por concursos. Es importante aclarar que existen concursos que se realizan periódicamente por lo que seria de agradecer que al introducir uno de estos concursos poner el año ya que el sistema no permitirá una entrada repetida (un ejemplo de convocatorias periódicas son los Premios de Arte Generación de Caja Madrid que se realizan anualmente).
Convocante: aquí se introduce qué entidad (fundación, organismo, empresa, etc…) realiza la convocatoria.
Disciplina: un campo desplegable donde se deve seleccionar a que tipo de disciplina va dirigida la convocatoria artística. Estas són: dibujo, escultura, fotografía, pintura, video, grabado, nuevas tecnologias, artes visuales, artes escénicas. Se han introducido denominaciones como artes visuales o artes escénicas que tienen un carácter mas general ya que es posible que se introduzcan convocatorias que no se adecuen a  ninguna de las disciplinas introducidas. En el caso de que la convocatoria que se introduzca no se adecue a ningun nombre es recomendable seleccionar uno de carácer general, y posteriormente contactar con el administrador explicando el caso, para que este pueda introducir un nuevo campo de selección con el nuevo nombre de disciplina.
Modalidad: aquí se deve seleccionar que modalidad de concurso es, que tipo de premio ofrece, si es una beca económica, si es una residencia, si lo que se ofrece es un espacio expositivo, etc… Si no hay ningún nombre que coincida con el tipo de modalidad contacta con el administrador y este añadirá el nuevo campo de selección.
Fecha entrega: aqui se añade un menu emergente con un calendario en el que se deve seleccionar la fecha de entrega de la convocatoria. Se pueden introducir convocatorias con fechas de entrega pasadas para introducir las convocatorias que han rechazado un proyecto artístico o convocatorias que són ahun activas, estas últimas aparecerán listadas en la sección de convocatorias activas que es una sección donde el usuario puede poner en común las convocatorias. Es muy importante poner la fecha correctamente para ya que dependiendo de si es una fecha pasada o próxima la convocatoria aparecerá en una sección o otra.
Descripción: aquí se introduce una breve descripción del concurso. Podiendose poner información sobre el jurado o sobre las bases. El premio en que consiste la convocatoria, etc…
Enlace: aquí se deve introducir el enlace a las bases del concurso, este campo es el único que no es obligatorio ya que si se introduce una convocatoria pasada dirigida a describir el concurso que ha rechazado un proyecto, posiblemente no exista el enlace web a las bases del concurso, ya que la fecha ha expirado.
